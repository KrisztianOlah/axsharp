# IxNodeVisitor class

Provides implementation of combined syntax-semantic three. Data object must implement [`ICombinedThreeVisitor`](../Ix.Compiler.Core/ICombinedThreeVisitor.md)'s methods to provide source generation. Not all Visit are implemented. We only visit nodes that have relevance for PLC's twin objects (Types, Variables etc).
