## Remarks

This method accesses the item in a single request over communication layer; accessing multiple items may result in performance degradation over the communication interface with the target system. This method is to be used only when you need the item to be read from the PLC and the return the control of the program. Consider using [`ITwinObject`](../../Ix.Connector/ITwinObject.md)s [`ReadAsync`](../../Ix.Connector/TwinObjectExtensions/ReadAsync.md) or [`WriteAsync`](../../Ix.Connector/TwinObjectExtensions/WriteAsync.md) methods for bulk access to entire structures.
